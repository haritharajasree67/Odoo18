Pos Auto Send Report

We can send POS reports automatically at a fixed time. The system will email the sales details, allowing daily sales to be accessed easily.

1.Navigate to the POS menu for quick and easy access.

2.Access settings by clicking on the three dots for configuration options.

3.Add the mail ids which the reports need to be received while closing the session.

4.In a fixed time you will receive a mail contain all the reports regarding the sale.
