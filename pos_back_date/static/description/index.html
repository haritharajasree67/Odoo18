POS Back Date Entry

Enhances Odoo Point of Sale by allowing users to backdate POS sessions and orders with ease. It helps maintain accurate accounting and reporting when sales need to be recorded for a previous date.

1.Navigate to the POS menu for quick and easy access.

2.Click on the Edit

3.Activate the Allow Back date in the configurations to backdate the orders.

4. The session open date is used for all orders created within the session, and all orders will be reflected on the same date. The session close date will also remain the same as the session open date.
