==============================
POS Auto Send Mail – Odoo 18
============================

.. image:: banner.png
:alt: POS Backdate Entry
:align: center


Overview
--------
This module enhances the Odoo Point of Sale by allowing POS sessions and orders
to be automatically created using a backdated session date. It ensures that all
POS entries are recorded accurately for the selected business date.

Key Features
------------
* Automatically creates POS sessions using a selected backdate
* All POS orders are recorded under the same session open date
* Session close date remains the same as the session open date
* Ensures accurate sales, inventory, and accounting reports
* Eliminates manual corrections for missed or delayed POS entries
* Fully compatible with Odoo POS workflow

Business Benefits
-----------------
This module is ideal for businesses that:
* Need to record POS sales for a previous date
* Want consistent session and order dates
* Require accurate historical sales reporting
* Want to avoid accounting mismatches caused by incorrect POS dates

Usage
-----
1. Select the desired backdate while opening the POS session
2. Create POS orders as usual
3. All orders will be recorded with the selected session date
4. Close the session; the close date will match the open date

Technical Notes
---------------
* No changes to existing POS workflows
* Works seamlessly with accounting and inventory modules
* Secure and reliable date handling
